# Welcome to Orion Docs

Orion is a powerful, locally-hosted AI assistant that gives you control over your models and data.

## Getting Started

### 1. Run the App
```bash
python launcher.py
```
This starts the TUI, which serves both the Frontend (Port 5173) and Backend (Port 8000).

### 2. View these Docs
```bash
mkdocs serve
```
Access the documentation at [http://localhost:8002](http://localhost:8002).

## Features
-   **Dual Model Support**: Use Google Gemini or Local Ollama.
-   **Rich Frontend**: A beautiful React 19 interface.
-   **TUI Launcher**: Manage everything from the terminal.
