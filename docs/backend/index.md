# Backend Overview

The Backend is the "Engine Room" of Orion.

## Directory Structure
-   **Core**: The brain logic (`Core`, `Server`).
-   **Agents**: Specialized sub-brains for file processing.
-   **Utils**: Support libraries (`Auth`, `Config`).

## Navigation
Use the sidebar to explore specific components.
-   **Core Logic**: How the Session Loop works.
-   **Agents**: How files are analyzed.
-   **Main Utils**: The Toolbox.
